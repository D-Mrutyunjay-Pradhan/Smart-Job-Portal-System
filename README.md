# Smart Job Portal System
## 📌 Project Overview

Smart Job Portal System is a web-based platform developed to simplify the recruitment and job application process for students, recruiters, and administrators. The project provides separate dashboards for Admin, Recruiter, and Student users with dedicated functionalities for managing jobs, candidates, profiles, reports, and user settings.

The system is designed using HTML, CSS, and JavaScript with a modern dashboard interface and responsive frontend pages.

# 🚀 Features
## 👨‍🎓 Student Module
* Student Registration & Login
* Student Dashboard
* Profile Management
* Account Settings
* View Job Opportunities
* User-Friendly Interface

## 🧑‍💼 Recruiter Module
* Recruiter Dashboard
* Post New Jobs
* Manage Job Listings
* Candidate Management
* Calendar Section
* Reports & Analytics
* Settings Management

## 🛠️ Admin Module
* Admin Dashboard
* Manage Students
* Manage Recruiters
* View Reports
* Admin Profile Management

## 🌐 Frontend Pages
* Home Page
* Login Page
* Registration Page
* Forgot Password Page
* Contact Us Page
* Job Portal Landing Page

# 🎯 Objectives of the Project
* To simplify the online recruitment process.
* To provide separate portals for students, recruiters, and admin.
* To improve communication between recruiters and job seekers.
* To create an organized dashboard system for managing job-related activities.
* To provide a modern and responsive user interface.

# 🔍 System Modules
1. Student Dashboard
Students can manage their profiles, settings, and explore job opportunities through an interactive dashboard.

2. Recruiter Dashboard
Recruiters can post jobs, manage candidates, generate reports, and organize recruitment activities.

3. Admin Dashboard
Admins have full control over students, recruiters, and reports for monitoring platform activities.
